- `<space>e` - Toggle file explorer
- `<space>o` - Focus file explorer

- `gr` - Go to references

- `<space>gg` - Open Lazygit

- `<space>tv` - Vertical terminal

- `K` - Hover documentation
- `<space>ca` - Code actions
- `<space>rn` - Rename symbol
